# VS Code Keyboard Shortcuts for Windows

Visual Studio Code has hundreds of keyboard shortcuts, and the full list depends on your installed extensions and customizations. The most comprehensive reference is the built-in PDF guide in VS Code itself. Here's how to access it:

1. Open VS Code.
2. Go to **Help > Keyboard Shortcut Reference** (or press **Ctrl+K Ctrl+S** to open the Keyboard Shortcuts editor, then click the "Open Keyboard Shortcuts (PDF)" button).

You can also download the platform-specific PDF directly:
- **Windows**: [Keyboard Shortcuts PDF](https://go.microsoft.com/fwlink/?linkid=832145)

For a quick overview, here are some of the most common and useful keyboard shortcuts for Windows (organized by category). These are based on VS Code's default settings (US English keyboard layout). Note that some may vary if you have extensions installed or have customized them.

## Tips
- Many shortcuts work in combination (e.g., **Ctrl+K** is a prefix for many commands).
- You can customize shortcuts via the Keyboard Shortcuts editor (**Ctrl+K Ctrl+S**).
- If a shortcut doesn't work, it might be overridden by an extension—check for conflicts.
- For multi-cursor editing, hold **Alt** and click, or use **Ctrl+Alt+Up/Down**.
- VS Code adapts shortcuts based on your keyboard layout, but the above assumes a standard US layout.
